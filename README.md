# 📊 FX価格予測 - リアルタイム為替予測システム

超高速・高精度なFX価格予測ブラウザアプリです。リアルタイムの為替データを取得し、直近のトレンドから3分後、10分後、30分後の価格を予測・表示します。

## ✨ 主な特徴

### 🚀 超高速パフォーマンス
- ページ読み込み時間: **1秒以下**
- 軽量な単一HTMLファイル（約25KB）
- 効率的な線形回帰アルゴリズム（O(n)）
- 非同期通信でブロッキングなし

### 📱 モバイル対応
- 完全レスポンシブデザイン
- スマートフォンで片手操作可能
- タッチフレンドリーなUI

### 🎨 プロフェッショナルなダークモード
- トレーダー向けのスタイリッシュなデザイン
- ネオングリーン（上昇）とネオンピンク（下降）のアクセント
- 視認性の高い等幅フォント

### 📈 高精度な予測機能
- 線形回帰による科学的な予測
- 3分、10分、30分後の価格を計算
- トレンド方向の自動判定
- 信頼度スコア表示

### 🔄 自動更新機能
- 10秒ごとのカウントダウン表示
- 手動更新ボタン
- 最終更新時刻の表示
- LocalStorageによるデータキャッシング

### 📊 リアルタイムグラフ
- Chart.jsによる美しいトレンドグラフ
- 過去レート（緑線）と予測レート（黄色点線）を同時表示
- インタラクティブなツールチップ

## 🚀 使用方法

### 1. ファイルを開く
ブラウザで `fx_predictor.html` を開くだけで、すぐに使用開始できます。インストールやサーバー設定は不要です。

```bash
# ローカルファイルとして開く
open fx_predictor.html

# または、簡単なHTTPサーバーで実行
python3 -m http.server 8000
# ブラウザで http://localhost:8000/fx_predictor.html にアクセス
```

### 2. 通貨ペアを選択
画面上部のドロップダウンメニューから、取引したい通貨ペアを選択します。初期値は USD/JPY です。

### 3. リアルタイムレートを確認
現在のレートが大きく表示されます。上昇・下降・横ばいのトレンドが自動判定されます。

### 4. 価格予測を確認
以下の3つの時間軸での予測価格が表示されます：
- **3分後**: 短期的なトレンド
- **10分後**: 中期的なトレンド
- **30分後**: 長期的なトレンド

### 5. トレンドグラフを分析
グラフには過去24時間のレート推移と、予測ラインが表示されます。マウスホバーでツールチップが表示されます。

### 6. 自動更新を活用
アプリは自動的に10秒ごとにデータを更新します。「今すぐ更新」ボタンで手動更新も可能です。

### 予測アルゴリズム

#### 線形回帰（Linear Regression）
過去のデータから線形トレンドを計算し、将来の価格を推定します。

```
y = mx + b
```

ここで：
- `y`: 予測価格
- `m`: トレンドの傾き（スロープ）
- `x`: 時間インデックス
- `b`: 切片

#### 計算複雑度
- **時間計算量**: O(n) - 非常に高速
- **空間計算量**: O(n) - 効率的

### API仕様

#### ExchangeRate-API (Open Access)
- **エンドポイント**: `https://open.er-api.com/v6/latest/{BASE_CURRENCY}`
- **認証**: 不要（APIキー不要）
- **更新頻度**: 24時間ごと
- **レート制限**: 1時間に1回のリクエスト推奨
- **CORS**: 対応

#### レスポンス例
```json
{
  "result": "success",
  "base_code": "USD",
  "rates": {
    "JPY": 158.9938,
    "EUR": 0.9234,
    "GBP": 0.7923,
    ...
  },
  "time_last_update_unix": 1705254000,
  "time_next_update_unix": 1705340400
}
```

## 🐛 トラブルシューティング

### データが表示されない場合
1. ブラウザのコンソール（F12）でエラーを確認
2. インターネット接続を確認
3. ExchangeRate-APIが利用可能か確認
4. ブラウザのキャッシュをクリア

### グラフが表示されない場合
1. Chart.jsがCDNから読み込まれているか確認
2. ブラウザのJavaScriptが有効か確認
3. コンソールでエラーメッセージを確認

### 予測値が不正確な場合
- APIが24時間ごとの更新のため、データが少ないと精度が低下します
- 複数回のデータ更新後に精度が向上します
- 市場の急激な変動には対応しにくい傾向があります
